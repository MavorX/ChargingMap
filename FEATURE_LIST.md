# VWer-Charging 功能清单说明文档

## 项目概述

VWer-Charging 是一款基于 Vue 3 + Pinia + TailwindCSS + 高德地图的电动汽车充电站查找与管理应用，为用户提供充电站搜索、预约、充电、支付、评价等一站式充电服务。

**技术栈：** Vue 3.5 + Pinia 2.3 + Vue Router 4.4 + TailwindCSS 3.4 (本地构建) + Font Awesome 6.5 (本地npm包) + 高德地图 JS SDK 2.0 + Vite 6.0

## 功能模块清单

### 一、地图与站点发现

| 功能 | 说明 | 状态 |
|------|------|------|
| 地图展示 | 基于高德地图 3D 视图，默认定位到合肥区域 | ✅ 可用 |
| 自动定位 | 页面加载时自动获取用户位置并居中地图，使用高德 Geolocation 插件 + useNative 优先GPS定位 | ✅ 可用 |
| 手动定位 | 点击 Header 定位按钮，使用高德 Geolocation 定位，useNative:true 优先浏览器GPS，失败时提供详细错误提示 | ✅ 可用 |
| 站点标记 | 充电站以圆形图标+闪电SVG+状态标签显示，颜色反映可用状态（绿色=有可用，红色=已满，灰色=维护中） | ✅ 可用 |
| 标记聚合 | 充电站标记支持聚合显示，聚合圈显示站点数量和可用数，hover 放大效果 | ✅ 可用 |
| 信息窗口 | 点击站点标记弹出信息窗口，包含站名、快慢充可用数、地址、价格和"导航前往"按钮（可点击交互） | ✅ 可用 |
| 点击查看详情 | 点击站点标记弹出信息窗口，点击"导航前往"按钮打开站点详情面板 | ✅ 可用 |
| 地图缩放 | 右下角浮动按钮支持地图放大/缩小，直接调用 AMap zoomIn/zoomOut API | ✅ 可用 |
| 站点导航 | 在信息窗口或站点详情中点击"导航"按钮，使用 AMap.Driving 驾车路线规划，自动绘制路线并显示距离和时间 | ✅ 可用 |

## 面板联动流程

```
地图点击站点标记 → 站点详情面板
站点列表点击站点 → 站点详情面板
站点详情 → 预约充电 → 预约成功提示
站点详情 → 充电设置 → 开始充电 → 充电中 → 结束充电 → 支付面板 → 支付成功 → 评价面板
Header 通知按钮 → 通知中心
Header 用户按钮 → 用户中心 → 通知中心
Header 定位按钮 → 地图定位到用户位置（高德 Geolocation 插件定位）
右下角筛选按钮 → 筛选面板
```

## 修复记录

### v10-final 修复内容

| 问题 | 修复方案 | 影响文件 |
|------|----------|----------|
| 图标大量无法加载 | Font Awesome 从 4.7.0 CDN 升级到 6.5.1 本地npm包，含 v4-shims 兼容层 | `main.js`, `index.html`, `ReservationPanel.vue` |
| 定位功能失效 | 使用 AMap.plugin 异步加载 Geolocation 插件，useNative:true 优先浏览器GPS定位，GeoLocationFirst:true 优先GPS而非IP | `mapService.js` |
| 定位精度偏差近百米 | 添加 useNative:true 使用浏览器原生GPS定位，maximumAge:0 不使用缓存，关闭 showCircle/showMarker 由代码手动添加精度圈 | `mapService.js` |
| 路线规划报错 INVALID_USER_SCODE | **根因：缺少安全密钥配置**。在 index.html 中添加 window._AMapSecurityConfig 配置 securityJsCode，必须在 AMap 脚本加载前设置 | `index.html` |
| 路线规划 AMap.Driving 构造失败 | 使用 AMap.plugin(['AMap.Driving'], callback) 异步加载插件，在回调内创建 Driving 实例并调用 search | `mapService.js` |
| 信息窗口无法关闭 | 信息窗口添加关闭按钮(×)，点击地图空白区域关闭信息窗口 | `mapService.js`, `MapView.vue` |
| 信息窗口"导航前往"进入详情页 | 改为派发 navigateToStation 事件直接开始驾车导航 | `mapService.js`, `HomeView.vue` |
| 站点筛选不支持组合 | 将筛选从单选改为多选 toggle，支持同时选择快充+可预约等组合条件 | `stations.js`, `StationsList.vue` |
| 站点标记UI不美观 | 重新设计标记为圆形图标+闪电SVG+状态标签，聚合圈增加 hover 放大效果 | `mapService.js` |
| 放大缩小按钮无响应 | 改用 Vue @click 直接调用 mapManager.zoomIn/zoomOut API | `HomeView.vue`, `mapService.js` |
| 充电设置滑钮超出范围 | 缩小 toggle 尺寸(11x6)，添加 flex-shrink-0/min-w-0 防止溢出 | `ChargingSettings.vue` |
| MapView.vue 重复代码 | 移除 MapView.vue 中重复的 doGeolocation 和 handleHeaderEvent | `MapView.vue` |
| TailwindCSS 双重配置 | 移除 CDN TailwindCSS 和内联配置，统一使用本地 TailwindCSS 构建 | `index.html`, `main.css` |
| 冗余数据目录 | 删除根目录 data/ 和 dist/ 构建输出（与 public/data/ 重复） | 已删除 |
